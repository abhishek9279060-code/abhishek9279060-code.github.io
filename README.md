# Image Compressor Tool

This is a simple and fast online image compressor tool.
You can reduce image size without losing much quality.

## How to Use
1. Click on the "Choose Image" button
2. Select an image from your device
3. Adjust compression level if available
4. Click on "Compress Image"
5. Download the compressed image

## Features
- Fast image compression
- No signup required
- Works on mobile and desktop
- Supports JPG, PNG, WebP

## Note
All image processing happens in your browser.
Your images are not uploaded to any server.
